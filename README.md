 
 # POP-OT

Code copy and adapted from [SOP+](https://github.com/shengliu66/SOP) and [ELR](https://github.com/shengliu66/ELR) repository.

The partial overparametrization loss class is [`partial_overparametrization_loss`](model/loss.py#L264)


To launch the training: 

```
python train.py -c [configuration file]
```
